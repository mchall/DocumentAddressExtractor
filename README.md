# Document Address Extractor
Attempts to find and extract an address in a document like a utility bill. 

<i>NOTE: Assumes a 3 line address in following format:</i>

<b>NAME</b>

<b>ADDRESS LINE 1</b>

<b>ADDRESS LINE 2</b>
